## 📚 Chapters

### 🌱 Beginner Chapters

> The foundation. Everything starts here — no prior experience needed.

<details>
<summary><b>📘 Chapter 01 — Introduction to Python</b></summary>
<br/>

- What is Python and why learn it?
- Installing Python and setting up VS Code
- Writing your very first Python program
- Understanding `print()` and comments
- Python syntax basics

</details>

<details>
<summary><b>📘 Chapter 02 — Variables & Data Types</b></summary>
<br/>

- What are variables and how to declare them
- Core data types: `int`, `float`, `str`, `bool`
- `type()` function to check data types
- Taking user input with `input()`
- Type casting: `int()`, `float()`, `str()`

</details>

<details>
<summary><b>📘 Chapter 03 — Operators</b></summary>
<br/>

- Arithmetic operators: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Comparison operators: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators: `and`, `or`, `not`
- Assignment operators: `=`, `+=`, `-=`, `*=`
- Bitwise operators: `&`, `|`, `^`, `~`, `<<`, `>>`

</details>

<details>
<summary><b>📘 Chapter 04 — Strings</b></summary>
<br/>

- String creation, indexing and slicing
- String methods: `upper()`, `lower()`, `strip()`, `split()`, `replace()`, `find()`
- String formatting: `%`, `.format()`, f-strings
- Multiline strings and escape characters
- String immutability

</details>

<details>
<summary><b>📘 Chapter 05 — Lists</b></summary>
<br/>

- Creating lists and accessing elements
- List methods: `append()`, `insert()`, `remove()`, `pop()`, `sort()`, `reverse()`
- List slicing and nested lists
- List comprehension
- Iterating over lists

</details>

<details>
<summary><b>📘 Chapter 06 — Tuples</b></summary>
<br/>

- Creating and accessing tuples
- Tuple packing and unpacking
- Tuple vs List — when to use which
- Tuple methods: `count()`, `index()`
- Immutability of tuples

</details>

<details>
<summary><b>📘 Chapter 07 — Sets</b></summary>
<br/>

- Creating sets and their properties (unordered, unique)
- Set methods: `add()`, `remove()`, `discard()`, `clear()`
- Set operations: union, intersection, difference, symmetric difference
- Frozen sets
- Set comprehension

</details>

<details>
<summary><b>📘 Chapter 08 — Dictionaries</b></summary>
<br/>

- Creating dictionaries and CRUD operations
- Dictionary methods: `keys()`, `values()`, `items()`, `get()`, `update()`
- Nested dictionaries
- Dictionary comprehension
- Looping through dictionaries

</details>

<details>
<summary><b>📘 Chapter 09 — Conditional Statements</b></summary>
<br/>

- `if`, `elif`, `else` statements
- Nested conditions
- Ternary / conditional expressions
- `match` statement (Python 3.10+)
- Truthy and falsy values

</details>

<details>
<summary><b>📘 Chapter 10 — Loops</b></summary>
<br/>

- `for` loop and `range()` function
- `while` loop
- `break`, `continue`, and `pass` statements
- Nested loops
- `else` clause with loops
- `enumerate()` and `zip()`

</details>

---

### ⚙️ Intermediate Chapters

> Where things get interesting. Real programming starts here.

<details>
<summary><b>📗 Chapter 11 — Functions</b></summary>
<br/>

- Defining functions with `def`
- Parameters, arguments, and return values
- Default and keyword arguments
- `*args` and `**kwargs`
- Scope: local vs global variables
- `lambda` (intro)

</details>

<details>
<summary><b>📗 Chapter 12 — Recursion</b></summary>
<br/>

- What is recursion?
- Base case and recursive case
- Classic examples: factorial, Fibonacci, binary search
- Stack overflow and recursion limits
- When to use recursion vs loops

</details>

<details>
<summary><b>📗 Chapter 13 — File Handling</b></summary>
<br/>

- Opening files: `open()` with modes (`r`, `w`, `a`, `rb`)
- Reading files: `read()`, `readline()`, `readlines()`
- Writing and appending to files
- `with` statement (context managers)
- Working with CSV files

</details>

<details>
<summary><b>📗 Chapter 14 — Exception Handling</b></summary>
<br/>

- Types of errors: syntax errors vs exceptions
- `try`, `except`, `else`, `finally` blocks
- Catching specific exceptions
- Raising exceptions with `raise`
- Creating custom exception classes

</details>

<details>
<summary><b>📗 Chapter 15 — OOP Basics</b></summary>
<br/>

- What is Object-Oriented Programming?
- Classes and objects
- `__init__()` constructor and `self`
- Instance variables and methods
- `__str__()` and `__repr__()` dunder methods

</details>

<details>
<summary><b>📗 Chapter 16 — OOP Advanced</b></summary>
<br/>

- Inheritance and `super()`
- Method overriding (Polymorphism)
- Encapsulation: private and protected members
- Abstraction with `ABC`
- Class methods, static methods, and properties
- Multiple inheritance

</details>

<details>
<summary><b>📗 Chapter 17 — Modules & Packages</b></summary>
<br/>

- Built-in modules: `os`, `sys`, `math`, `random`, `datetime`
- Creating your own modules
- Packages and `__init__.py`
- Installing third-party packages with `pip`
- Virtual environments with `venv`

</details>

---

### 🚀 Advanced Chapters

> Pythonic mastery. This is what separates good from great.

<details>
<summary><b>📕 Chapter 18 — Iterators & Generators</b></summary>
<br/>

- Iterables vs iterators
- `iter()` and `next()` functions
- Creating custom iterator classes
- Generator functions with `yield`
- Generator expressions
- `itertools` module

</details>

<details>
<summary><b>📕 Chapter 19 — Decorators & Closures</b></summary>
<br/>

- First-class functions and closures
- Understanding and creating decorators
- Decorators with arguments
- `functools.wraps` to preserve metadata
- Chaining multiple decorators
- Class-based decorators

</details>

<details>
<summary><b>📕 Chapter 20 — Advanced Python</b></summary>
<br/>

- `lambda` functions deep dive
- `map()`, `filter()`, `reduce()`
- List, dict, and set comprehensions
- `zip()`, `enumerate()`, `any()`, `all()`
- Unpacking operators `*` and `**`
- Walrus operator `:=`

</details>

<details>
<summary><b>📕 Chapter 21 — Regular Expressions</b></summary>
<br/>

- What are regex patterns?
- `re` module: `match()`, `search()`, `findall()`, `sub()`
- Special characters: `.`, `*`, `+`, `?`, `^`, `$`
- Character classes and groups
- Flags: `re.IGNORECASE`, `re.MULTILINE`
- Real-world regex use cases

</details>

## 🔥 How to Use This Repo

<div align="center">

```
Step 1 → Pick a chapter in order (don't skip!)
Step 2 → Read the PDF notes carefully
Step 3 → Close the notes and attempt the practice set
Step 4 → Stuck? Try harder for 10 more minutes
Step 5 → Then check the solutions and understand WHY
Step 6 → Move to the next chapter 🚀
```

</div>

<br/>

> 💡 **Pro Tip:** Don't just read the solutions. Type every solution yourself — muscle memory is how programming skills are built.

> ⚠️ **Don't skip chapters.** Every chapter builds on the previous one. Skipping basics causes confusion later.

> 🔁 **Revise regularly.** Go back to old chapters when you're stuck — the answer is usually there.
